Action Professionnelle 2ème Année

Sur cette page vous trouverez un de mes projet realisé durant ma deuxième année de BTS SIO sur Windows Server et Active Directory .

Windows Server & Active Directory

Création d'une VM Windows Server

1.1 Installation de la Vm via Oracle Virtual Box

Installation de Windows Server

Le premier objectif de notre projet était de créer une machine virtuelle sous Windows Server à l’aide de VirtualBox. Pour cela, nous avons généré une VM, installé le système d’exploitation, puis configuré le réseau en mode « Accès par pont » afin de permettre à la VM de communiquer avec le réseau local.

Ensuite, nous avons effectué les configurations post-installation :

  • Installation des Additions invité pour améliorer les performances et l’affichage.
  • Mise à jour de Windows via Windows Update afin de sécuriser et stabiliser le système.

1.2 Installation et configuration d'Active Directory

Une fois Windows installé et prêt, nous avons ouvert le Gestionnaire de serveur pour ajouter le rôle AD DS (Active Directory Domain Services).

Étapes principales :
  • Ajout du rôle AD DS
  • Promotion du serveur en contrôleur de domaine
  • Création d’une nouvelle forêt
  • Vérification via la commande dcdiag pour s’assurer que le serveur fonctionne sans erreur
Résultat attendu :

La commande ipconfig permet de vérifier la bonne configuration du réseau, et un screenshot a été pris pour conserver une trace du bon fonctionnement.
